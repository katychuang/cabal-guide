setup

Download cabal for OSX

Download cabal for Windows

Download cabal for Linux

create a cabal sandbox

create a new directory, open it and perform a
cabal sandbox init
to create a new cabal sandbox.

add dependencies

create a copy of a package by running the command
cabal install --only-dependencies

build

your sandbox consists of specific version of packages you've added and can be build with
cabal build

useful hints

attaching patched or an unreleased version of a library
cabal sandbox add-source
delete sandbox
cabal sandbox delete
alternative manual clean
rm -rf .cabal-sandbox cabal.sandbox.config
